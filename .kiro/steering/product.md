# Product Overview

The Online Retail Simulator is a lightweight Python package for generating synthetic retail data for experimentation and causal inference in e-commerce contexts.

## Core Purpose
- Generate realistic product catalogs and daily sales transactions
- Support reproducible experiments with random seeds
- Enable testing and demos without exposing real business data
- Provide educational tool for data science concepts

## Key Features
- Two simulation modes: rule-based (simple) and synthesizer-based (ML-powered with SDV)
- JSON configuration-driven workflow
- Deterministic output with seed control
- Simple product generation across 8 categories
- Realistic daily sales patterns with configurable probability
- JSON export for integration with other tools

## Target Users
- Data scientists and researchers
- E-commerce analysts
- Educators teaching data science concepts
- Developers needing synthetic retail data for testing
