# Retail Catalog Simulator

A lightweight Python package for simulating realistic product data and business metrics for teaching and evaluating causal inference strategies in e-commerce contexts.

## Overview

The Retail Catalog Simulator generates synthetic but realistic data that mimics the complexity of real-world catalog systems, including product attributes, sales metrics, pricing dynamics, and causal relationships between interventions and outcomes.

## Installation

```bash
pip install retail-catalog-simulator
```

For development:

```bash
pip install -e ".[dev]"
```
